# ComicParse:i漫
### 简介
这是一个第三方的漫画阅读APP.所有数据均来自中国国内各大APP(台湾属于中国),致谢列表在下面.

本项目仅用于研究分析学习使用,他人盗取代码二次修改打包引起的版权问题与各项后果与作者本人(秋城落叶)无关.

许可证使用MIT授权.
